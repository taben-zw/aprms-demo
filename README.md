# APRMS Demo

A fast demo of an Automated Parking Revenue Management System for the City of Bulawayo.

## Features
- Mock ANPR vehicle identification
- Mock CVR lookup
- Automated billing by vehicle class
- Cashless tap-card payment simulation
- Receipt generation
- Dashboard and transaction logs

## Tariffs
- Small Vehicle: $1
- Combi: $2
- Sprinter: $4
- Mini Bus: $5
- Bus: $10

## Run locally
```bash
npm install
npm start
```

Open:
http://localhost:3000
